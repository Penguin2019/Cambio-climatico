# Cambio-climatico
Proyecto de cambio climatico
EcoPingu es una plataforma diseñada para compartir el conocimiento sobre contaminación y cuidado ambiental, además cuenta con un bot privado de Discord capaz de realizar diversas funciones para responder dudas del usuario relacionadas al tema.
EcoPingu es un bot de Discord capaz de:
🐧🌱Obtener datos meteorológicos de ciudades.
🐧🌱Obtener tips sobre el cuidado ambiental.
🐧🌱Buscar empresas de productos sustentables.
🐧🌱Buscar manualidades de materiales reciclables.
🐧🌱Obtener definiciones sobre temas ambientales.
🐧🌱Preguntar por la huella de carbono.
